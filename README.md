# 🚀 Python Project Coach

An interactive dashboard built with **Streamlit** designed to help beginner programmers level up their coding skills! This application acts as an automated mentor—checking project file structures, flagging bugs, calculating letter grades, and providing friendly AI-driven feedback using the **Groq Cloud API** and Llama 3.1.

---

## ✨ Features
* 📁 **Structure Analyzer:** Automatically maps out project file structures to ensure best practices.
* 🐛 **Bug Detection:** Flags common logic flaws, syntax issues, and performance bottlenecks.
* 📊 **Automated Scoring:** Generates a real-time letter grade based on code quality metrics.
* 🤖 **Llama 3.1 Mentor:** Features an interactive AI chat window for personalized, step-by-step guidance on how to fix your code.

---

## 🛠️ Tech Stack
* **Frontend/Dashboard:** Streamlit
* **LLM Engine:** Groq Cloud API (Llama 3.1 model)
* **Language:** Python 3.11+
